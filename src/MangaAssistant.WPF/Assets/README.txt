The image should be:
- Approximately 350x500 pixels
- Dark-themed to match the UI
- Simple design with text indicating it's a placeholder

You can create this image using any image editing software or download a placeholder from a site like:
- https://placehold.co/350x500/1e1e1e/ffffff.jpg

After adding the image, make sure to:
1. Set its Build Action to "Resource" in Visual Studio
